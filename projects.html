Projects

A collection of my current and past projects for display.

ProjectPal

A project idea generator and manager made using Javascript and the Electron library. The application is Designed to assist in the development of ideas for software. The app features user authentication using firebase and persistent data storage among other things.

Child Computer Activity Montior (CCAM)

The CCAM was the second-place project winner a the district-wide Klein Hacks hackathon for Klein high schools. The purpose of the project is to provide a simple but effective monitoring tool for parents to monitor and control their children's computer activity.

CodeDocs

A live updating code editor inspired by the collaborative editing ability of Google Docs. The application employs the use of Socket.io, a websocket framework that allows the communication from your application's front end to your server and the ability to pass data in between. The project is intended to provide a useful tool for small teams that want to share their ideas in code without having to manage git repositories. The application is currently deployed to Heroku, and is accessible here: CodeDocs
